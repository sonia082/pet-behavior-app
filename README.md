# 🐾 Pet Emotion Classifier

## About
This ML model classifies pet emotions into 3 categories:
- 😊 Happy
- 😔 Sad  
- 😠 Angry

  live open:
  https://2rnpiungh42njncegpwoxb.streamlit.app/

Based on input features like age, breed, activity level, and vocalization patterns.

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas


## Model Accuracy
89%
